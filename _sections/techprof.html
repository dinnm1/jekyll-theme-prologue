Navigation: George, Connor and I were experimenting on an effective design in class, however, that did not pan out. Later that week Connor attempted to create a navigation from scratch using the classes available in Foundation. It was functional, however incomplete and when I went into to work on it, it was extremely difficult to understand the code. To unburden future developers of any grief, I decided to re-do the navigation. I went with the simple route and used one of the many navigation building blocks available on the Foundation website and referenced that in the code for future devs and added the respective links to the pages on our website.


HTML -
<nav class="hover-underline-menu" data-menu-underline-from-center>

CSS -
$menu-underline-from-center-bg: #2C3840;
$menu-underline-from-center-border-thickness: 0.125rem;

.hover-underline-menu {
  width: 100%;

  .menu {

      color: $white;

    .underline-from-center {
      position: relative;

      &::after {
        content: "";

      }

      &:hover::after {
        left: 0;
        right: 0;

Having done a lot of CSS in the past, only now have I learned how to inherit css from other classes, this was very important as I needed to be able to customise the CSS to suit specific elements, at the same time working with Foundation CSS. For example, here I am extending the button class from Foundation and adding my own CSS to the element this class belongs to:

.button.search
{
    float: right;
    width: 40%;
}

As we were creating the modals and data display for every sea lion dynamically using multiple different functions, some buttons had to be created and appended before other elements. However, the team wanted certain buttons to appear at the very bottom of each column. To accomplish this I would normal use this CSS:

.button.submit_update
{
    position: absolute;
    bottom: 0;
}

However, that would place the button at the bottom right of the screen, whereas I needed it to be place at the bottom right of the div. It was extremely difficult to find any information on doing this within a div. Eventually I found this post that explained that I needed to add a relative position for the div that the button belongs to, like this:

.column_one
{
    position: relative;
}

.button.submit_update
{
    position: absolute;
    bottom: 0;
}
